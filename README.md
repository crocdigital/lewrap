## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.





## Introduction
LeWrap's website is a static site built in Astro and deployed via CloudCannon through Cloudflare. Files are hosted via git.

## Tech stack
- Astro
- Tailwind CSS
- CloudCannon (for CMS)
- Cloudflare
