## Introduction
LeWrap's website is a static site built in Astro and deployed via CloudCannon through Cloudflare. Files are hosted via git.

## Tech stack
- Astro
- Tailwind CSS
- CloudCannon (for CMS)
- Cloudflare

## NPM dependancies
- @astrojs/react (^4.4.2)
- @astrojs/vercel (^9.0.2)
- @tailwindcss/vite (^4.1.18)
- astro (^5.16.6)
- react (19.2.0)
- react-dom (19.2.0)
- swiper (^12.0.3)
